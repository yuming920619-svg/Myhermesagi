# Special Topic Discussion

這個資料夾保存「專題討論 / Special Topic Discussion」課程的觀眾評分表草稿。

用途：
- 根據講者姓名與報告主題，自動生成貼到 Google 表單的評分文字草稿
- 每位講者包含 5 個固定評分面向
- 內容會以 Markdown 檔保存，方便追蹤與版本控管

檔名規則：
- `YYYY-MM-DD_觀眾評分表草稿.md`
- 同一天的第二版：`YYYY-MM-DD_觀眾評分表草稿_第二版.md`

固定評分面向：
1. 研究內容與學術深度
2. 跨領域解釋能力
3. 投影片架構與邏輯組織
4. 表達技巧與專業態度
5. 問答回應與臨場反應

生成規則摘要：
- 預設建議分數：4/5
- 每則評論以繁體中文撰寫
- 每則評論至少 30 字，理想 32–45 字
- 若使用者提供投影片或報告照片，應先根據照片補足脈絡，再生成更貼題的評論
- 預設完成草稿後推送到 GitHub，除非使用者明確表示不要 push
